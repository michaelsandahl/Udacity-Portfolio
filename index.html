Portfolio Project

Creating a portfolio from a design mockup was the first project through Udacity. I used Bootstrap as the framework, and created the HTML and custom CSS to replicate the design. The exercise gave me some initial experience in taking a design concept to a working model by tweaking and testing the code as I went along, trying to approximate the initial design as best I could.
